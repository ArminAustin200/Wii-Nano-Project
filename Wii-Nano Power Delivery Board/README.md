# Wii-Nano PD Board
Features:
- Designed around the [IP2721] (https://datasheet.lcsc.com/lcsc/2006111335_INJOINIC-IP2721_C603176.pdf) IC
- USB Type C input
- 15V DC Output (Over USB Power Delivery Standard)
- USB 2.0 data transfer
- Rated for 15V 3A
